# Intake IN-014

Replace npm commander dependency with in-house lightweight CLI parser module. Commander v14 is the only runtime dependency and its API surface used is modest (Command, option, requiredOption, argument, action, hook, parseAsync, subcommand nesting, --help, version). Removing it eliminates the external dependency entirely, reduces supply-chain surface, and gives full control over CLI behavior.
